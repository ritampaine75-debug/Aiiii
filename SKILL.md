# রসায়ন - মোল, আয়তন এবং অণুর সংখ্যা: সম্পূর্ণ গাইড

---

## 📚 মূল সূত্রসমূহ

### 1. মোল সংখ্যা নির্ণয়ের সূত্র

```
মোল (n) = ভর (mass) / আণবিক ভর (Molar mass)

n = m / M
```

**যেখানে:**
- n = মোল সংখ্যা
- m = পদার্থের ভর (গ্রাম)
- M = পদার্থের আণবিক ভর (g/mol)

---

### 2. অণুর সংখ্যা নির্ণয়ের সূত্র

```
অণুর সংখ্যা (N) = মোল সংখ্যা × অ্যাভোগাড্রো সংখ্যা

N = n × Nₐ

বা সরাসরি: N = (m / M) × Nₐ
```

**যেখানে:**
- N = অণুর সংখ্যা
- n = মোল সংখ্যা
- Nₐ = অ্যাভোগাড্রো সংখ্যা = 6.022 × 10²³

---

### 3. গ্যাসের আয়তন নির্ণয়ের সূত্র (STP-তে)

```
STP-তে 1 মোল গ্যাস = 22.4 L আয়তন দখল করে

V = n × 22.4

বা সরাসরি: V = (m / M) × 22.4
```

**যেখানে:**
- V = গ্যাসের আয়তন (লিটার)
- n = মোল সংখ্যা
- M = আণবিক ভর (g/mol)
- 22.4 L/mol = STP-তে মোলার আয়তন

---

### 4. গ্যাসের আণবিক ভর নির্ণয়ের সূত্র (STP-তে)

```
আণবিক ভর (M) = (ভর × 22.4) / আয়তন

M = (m × 22.4) / V
```

**বা:**
```
আণবিক ভর (M) = ভর / মোল সংখ্যা

M = m / n
```

---

### 5. অণুর সংখ্যা থেকে পরমাণুর সংখ্যা

```
মোট পরমাণুর সংখ্যা = অণুর সংখ্যা × প্রতিটি অণুতে পরমাণুর সংখ্যা

টোটাল atoms = N × (পরমাণু সংখ্যা)
```

---

### 6. ঘনত্ব থেকে ভর নির্ণয়ের সূত্র

```
ভর (m) = ঘনত্ব (ρ) × আয়তন (V)

m = ρ × V
```

**যেখানে:**
- m = ভর (গ্রাম)
- ρ = ঘনত্ব (g/mL বা g/cm³)
- V = আয়তন (mL বা cm³)

---

# ✅ সমস্ত প্রশ্নের সমাধান

## প্রশ্ন 1: 46g সোডিয়ামের মোল সংখ্যা নির্ণয় কর

### দেওয়া:
- Na এর ভর = 46 g
- Na এর আণবিক ভর (M) = 23 g/mol

### সমাধান:

**ব্যবহৃত সূত্র:**
```
n = m / M
```

**গণনা:**
```
n = 46 / 23
n = 2 মোল
```

### উত্তর: **2 মোল** ✓

---

## প্রশ্ন 2: জলের ঘনত্ব 1g/mL হলে 3.6 mL জলে অণুর সংখ্যা গণনা কর

### দেওয়া:
- জলের ঘনত্ব (ρ) = 1 g/mL
- জলের আয়তন (V) = 3.6 mL
- H₂O এর আণবিক ভর (M) = 18 g/mol
- Nₐ = 6.022 × 10²³

### সমাধান:

**ধাপ 1: ভর নির্ণয় করি**
```
m = ρ × V
m = 1 × 3.6
m = 3.6 g
```

**ধাপ 2: মোল সংখ্যা নির্ণয় করি**
```
n = m / M
n = 3.6 / 18
n = 0.2 মোল
```

**ধাপ 3: অণুর সংখ্যা নির্ণয় করি**
```
N = n × Nₐ
N = 0.2 × 6.022 × 10²³
N = 1.2044 × 10²³
N ≈ 1.20 × 10²³ অণু
```

### উত্তর: **1.20 × 10²³ অণু** ✓

---

## প্রশ্ন 3: NH₃ গ্যাসের একটি অণুর ভর নির্ণয় কর

### দেওয়া:
- NH₃ এর আণবিক ভর (M) = 17 g/mol
  - (N = 14, H = 1, সুতরাং 14 + 3×1 = 17)
- Nₐ = 6.022 × 10²³

### সমাধান:

**ব্যবহৃত সূত্র:**
```
একটি অণুর ভর = আণবিক ভর / অ্যাভোগাড্রো সংখ্যা

একটি অণুর ভর = M / Nₐ
```

**গণনা:**
```
একটি অণুর ভর = 17 / (6.022 × 10²³)
            = 2.82 × 10⁻²³ g
            = 2.82 × 10⁻²⁶ kg
```

### উত্তর: **2.82 × 10⁻²³ গ্রাম** ✓

---

## প্রশ্ন 4: STP-তে 6.4 g ভরের একটি গ্যাসের আয়তন 2.24 L হলে গ্যাসটির আণবিক ভর কত?

### দেওয়া:
- গ্যাসের ভর (m) = 6.4 g
- গ্যাসের আয়তন (V) = 2.24 L (STP-তে)
- STP-তে মোলার আয়তন = 22.4 L/mol

### সমাধান:

**পদ্ধতি 1: মোল সংখ্যা থেকে**

```
ধাপ 1: মোল সংখ্যা নির্ণয়
n = V / 22.4
n = 2.24 / 22.4
n = 0.1 মোল

ধাপ 2: আণবিক ভর নির্ণয়
M = m / n
M = 6.4 / 0.1
M = 64 g/mol
```

**পদ্ধতি 2: সরাসরি সূত্র থেকে**

```
M = (m × 22.4) / V
M = (6.4 × 22.4) / 2.24
M = 143.36 / 2.24
M = 64 g/mol
```

### উত্তর: **64 g/mol** ✓

**এটি সম্ভবত নাইট্রিক অ্যাসিড (HNO₃) বা অন্য কোনো পদার্থ হতে পারে।**

## প্রশ্ন 5: STP-তে 11.2L অ্যামোনিয়া গ্যাসে অণুর সংখ্যা এবং মোট পরমাণুর সংখ্যা গণনা কর

### দেওয়া:
- NH₃ এর আয়তন (V) = 11.2 L (STP-তে)
- NH₃ এর আণবিক ভর (M) = 17 g/mol
- STP-তে মোলার আয়তন = 22.4 L/mol
- Nₐ = 6.022 × 10²³
- NH₃ এ পরমাণু সংখ্যা = 4 (1N + 3H)

### সমাধান:

**ধাপ 1: মোল সংখ্যা নির্ণয়**
```
n = V / 22.4
n = 11.2 / 22.4
n = 0.5 মোল
```

**ধাপ 2: অণুর সংখ্যা**
```
অণুর সংখ্যা = n × Nₐ
            = 0.5 × 6.022 × 10²³
            = 3.011 × 10²³ অণু
            ≈ 3.01 × 10²³ অণু
```

**ধাপ 3: মোট পরমাণুর সংখ্যা**
```
প্রতিটি NH₃ অণুতে পরমাণু = 1 + 3 = 4

মোট পরমাণু = অণুর সংখ্যা × 4
            = 3.011 × 10²³ × 4
            = 1.2044 × 10²⁴ পরমাণু
            ≈ 1.20 × 10²⁴ পরমাণু
```

### উত্তর:
- **অণুর সংখ্যা: 3.01 × 10²³** ✓
- **পরমাণুর সংখ্যা: 1.20 × 10²⁴** ✓

---

## প্রশ্ন 6: STP-তে 2.2 g কার্বন ডাইঅক্সাইড গ্যাসের আয়তন নির্ণয় কর

### দেওয়া:
- CO₂ এর ভর (m) = 2.2 g
- CO₂ এর আণবিক ভর (M) = 44 g/mol
  - (C = 12, O = 16, সুতরাং 12 + 2×16 = 44)
- STP-তে মোলার আয়তন = 22.4 L/mol

### সমাধান:

**ধাপ 1: মোল সংখ্যা নির্ণয়**
```
n = m / M
n = 2.2 / 44
n = 0.05 মোল
```

**ধাপ 2: আয়তন নির্ণয়**
```
V = n × 22.4
V = 0.05 × 22.4
V = 1.12 L
```

**সরাসরি সূত্র:**
```
V = (m / M) × 22.4
V = (2.2 / 44) × 22.4
V = 0.05 × 22.4
V = 1.12 L
```

### উত্তর: **1.12 লিটার** ✓

# 📋 সূত্রের বিস্তারিত ব্যাখ্যা

---

## 1️⃣ মোল সংখ্যা নির্ণয়ের সূত্র: n = m / M

### ব্যাখ্যা:

**মোল কি?**
- মোল হল পদার্থের পরিমাণের একটি একক
- 1 মোল = 6.022 × 10²³ কণা (অণু, পরমাণু, ইলেকট্রন ইত্যাদি)

**সূত্রের অর্থ:**
- যদি কোনো পদার্থের ভর জানো এবং তার আণবিক ভর জানো
- তাহলে মোল সংখ্যা = ভর ÷ আণবিক ভর

**উদাহরণ:**
- 46g সোডিয়াম = 46 ÷ 23 = 2 মোল
- এর মানে 46g সোডিয়ামে 2 মোল বা 2 × 6.022 × 10²³ = 1.2 × 10²⁴ পরমাণু আছে

**কখন ব্যবহার করবে:**
- যখন ভর দেওয়া আছে এবং মোল বা অণুর সংখ্যা বের করতে হবে

---

## 2️⃣ অণুর সংখ্যা: N = n × Nₐ

### ব্যাখ্যা:

**অণুর সংখ্যা কি?**
- একটি পদার্থে মোট কত সংখ্যক অণু আছে তা নির্দেশ করে

**সূত্রের অর্থ:**
```
অণুর সংখ্যা = মোল সংখ্যা × (প্রতিটি মোলে অণুর সংখ্যা)
           = n × Nₐ
           = n × 6.022 × 10²³
```

**উদাহরণ:**
- 0.2 মোল জল
- অণুর সংখ্যা = 0.2 × 6.022 × 10²³ = 1.2044 × 10²³

**কখন ব্যবহার করবে:**
- যখন মোল সংখ্যা জানো এবং কত অণু আছে তা বের করতে হবে

**সরাসরি সূত্র:**
```
N = (m / M) × Nₐ

এই সূত্রে সরাসরি ভর দিয়ে অণুর সংখ্যা বের করতে পারো
```

---

## 3️⃣ গ্যাসের আয়তন (STP): V = n × 22.4

### ব্যাখ্যা:

**STP কি?**
- STP = Standard Temperature and Pressure
- তাপমাত্রা = 0°C বা 273 K
- চাপ = 1 atm বা 101.325 kPa

**গুরুত্বপূর্ণ তথ্য:**
- STP-তে প্রতিটি মোল গ্যাস ঠিক 22.4 লিটার আয়তন দখল করে
- এটি সব গ্যাসের জন্য প্রযোজ্য (আদর্শ গ্যাসের জন্য)

**সূত্রের অর্থ:**
```
আয়তন (L) = মোল সংখ্যা × 22.4 L/mol
```

**উদাহরণ:**
- 0.5 মোল NH₃ গ্যাস
- আয়তন = 0.5 × 22.4 = 11.2 L

**সরাসরি সূত্র:**
```
V = (m / M) × 22.4

এই সূত্রে সরাসরি ভর দিয়ে আয়তন বের করতে পারো
```

**কখন ব্যবহার করবে:**
- শুধুমাত্র STP শর্তে গ্যাসের আয়তন বের করতে

---

## 4️⃣ আণবিক ভর: M = (m × 22.4) / V

### ব্যাখ্যা:

**আণবিক ভর কি?**
- একটি অণুর ভরকে পারমাণবিক ভর একক (u) তে পরিমাপ করা হয়
- কিন্তু g/mol এ প্রকাশ করা হয়

**সূত্রের উৎপত্তি:**
```
আমরা জানি: n = V / 22.4  এবং  M = m / n

তাই: M = m / (V / 22.4)
     M = m × 22.4 / V
```

**সূত্রের অর্থ:**
- যদি একটি গ্যাসের ভর এবং আয়তন জানো (STP-তে)
- তাহলে আণবিক ভর বের করতে পারো

**উদাহরণ:**
- 6.4g গ্যাস, 2.24L আয়তন
- M = (6.4 × 22.4) / 2.24 = 64 g/mol

**কখন ব্যবহার করবে:**
- যখন গ্যাসের ভর এবং আয়তন জানো, কিন্তু পদার্থ চিনতে পারো না

---

## 5️⃣ পরমাণুর সংখ্যা

### ব্যাখ্যা:

**পরমাণু vs অণু:**
- একটি অণু এক বা একাধিক পরমাণু দিয়ে তৈরি
- N₂ = 1টি অণুতে 2টি পরমাণু
- CO₂ = 1টি অণুতে 3টি পরমাণু (1C + 2O)
- NH₃ = 1টি অণুতে 4টি পরমাণু (1N + 3H)

**সূত্র:**
```
মোট পরমাণুর সংখ্যা = অণুর সংখ্যা × প্রতিটি অণুতে পরমাণুর সংখ্যা

টোটাল atoms = N × (এটম সংখ্যা)
```

**উদাহরণ:**
```
11.2L NH₃ গ্যাস
- অণুর সংখ্যা = 3.01 × 10²³
- প্রতিটি NH₃ তে 4টি পরমাণু
- মোট পরমাণু = 3.01 × 10²³ × 4 = 1.20 × 10²⁴
```

**কখন ব্যবহার করবে:**
- যখন মোট পরমাণুর সংখ্যা বের করতে হবে

---

## 6️⃣ ঘনত্ব থেকে ভর: m = ρ × V

### ব্যাখ্যা:

**ঘনত্ব কি?**
- ঘনত্ব = আয়তনের প্রতি একক ভর
- ঘনত্ব (ρ) = ভর / আয়তন
- একক: g/mL, g/cm³, kg/L ইত্যাদি

**সূত্রের অর্থ:**
```
ভর = ঘনত্ব × আয়তন
```

**উদাহরণ:**
- জলের ঘনত্ব = 1 g/mL
- 3.6 mL জলের ভর = 1 × 3.6 = 3.6g

**কখন ব্যবহার করবে:**
- যখন তরল বা কঠিনের আয়তন এবং ঘনত্ব জানো
- এবং ভর বের করতে হবে

### পদক্ষেপ 3: যোগ্য সূত্র নির্বাচন করো

| দেওয়া | খুঁজছো | সূত্র |
|--------|---------|--------|
| m, M | n | n = m/M |
| m, M | N (অণু) | N = (m/M) × Nₐ |
| n | V (STP) | V = n × 22.4 |
| m, M | V (STP) | V = (m/M) × 22.4 |
| m, V (STP) | M | M = (m × 22.4)/V |
| V (STP), M | m | m = (V × M)/22.4 |
| N (অণু) | n | n = N/Nₐ |
| ρ, V | m | m = ρ × V |
