# 🎓 সূত্রগুলির বিস্তারিত ব্যাখ্যা

## ১. মোল সংখ্যা: n = m / M

### কী, কেন এবং কীভাবে:

**কী হল মোল?**
- মোল হল পরিমাণের একটি মৌলিক একক (যেমন ডজন = 12)
- 1 মোল = ঠিক 6.022 × 10²³ কণা (যেকোনো কণা - অণু, পরমাণু, ইলেকট্রন)
- এটি রসায়নের ভাষা - বিজ্ঞানীরা ব্যবহার করেন

**কেন এই সূত্র?**
```
মোল = মোট কণার সংখ্যা / প্রতিটি মোলে কণার সংখ্যা
    = গ্রাম / (1 গ্রাম পার্টিকেলের ভর × Nₐ)
    = গ্রাম / (পারমাণবিক ভর সংখ্যা)
    = m / M
```

**কীভাবে ব্যবহার করবে?**
```
পদক্ষেপ 1: পদার্থের ভর জানো
পদক্ষেপ 2: এর আণবিক ভর খুঁজে বের করো (পারমাণবিক ভর যোগ করে)
পদক্ষেপ 3: ভর ÷ আণবিক ভর = মোল
```

**উদাহরণ:**
```
46g সোডিয়াম (Na)
- Na এর আণবিক ভর = 23 g/mol
- মোল = 46 / 23 = 2 মোল
- অর্থ: 46g সোডিয়ামে 2 মোল বা 2 × 6.022×10²³ = 1.204×10²⁴ পরমাণু আছে
```

**দৈনন্দিন উপমা:**
```
যদি 1 ডজন ডিম = 12 ডিম
তাহলে 1 মোল অণু = 6.022×10²³ অণু

ঠিক যেমন:
- 24টি ডিম = 2 ডজন
- 12.044×10²³ অণু = 2 মোল
```

## ২. অণুর সংখ্যা: N = n × Nₐ

### অণু গণনা করার সূত্র

**কী বোঝায়?**
- অণুর সংখ্যা = কত খানি অণু আছে?
- আমরা জানি 1 মোল = 6.022×10²³ অণু
- তাই যদি 2 মোল থাকে = 2 × 6.022×10²³ = 1.204×10²³ অণু

**সূত্র বুঝি:**
```
N = মোল × 6.022×10²³
  = (ভর / আণবিক ভর) × 6.022×10²³
  = (m/M) × Nₐ
```

**সরাসরি সূত্র (মোল ছাড়াই):**
```
N = (m / M) × Nₐ

এটি ব্যবহার করে সরাসরি ভর থেকে অণুর সংখ্যা বের করো
```

**উদাহরণ:**
```
3.6g জল (H₂O) তে কত অণু?
- H₂O এর আণবিক ভর = 18
- প্রথমে মোল: 3.6 / 18 = 0.2 মোল
- তারপর অণু: 0.2 × 6.022×10²³ = 1.2044×10²³

বা সরাসরি:
- N = (3.6/18) × 6.022×10²³ = 1.2044×10²³ অণু
```

**মানসিক ছবি:**
```
তুমি জানো: 1 মোল = ভারী ভারী সংখ্যক কণা
তাই: বেশি মোল = বেশি কণা
     কম মোল = কম কণা
```

## ৩. গ্যাসের আয়তন (STP): V = n × 22.4

### গ্যাসের অনন্য বৈশিষ্ট্য

**STP কি?**
- STP = Standard Temperature and Pressure
- তাপমাত্রা = 0°C (273 K)
- চাপ = 1 atম (মানক বায়ুচাপ)

**জাদু সংখ্যা: 22.4 L/mol**
```
STP-তে এটি সত্য:
- 1 মোল যেকোনো গ্যাস = ঠিক 22.4 লিটার আয়তন

কেন?
- আদর্শ গ্যাস সমীকরণ: PV = nRT
- P, T, R ধ্রুবক (STP-তে), তাই V ∝ n
- গণনা করলে: V = 22.4n
```

**ব্যবহার:**
```
পদক্ষেপ 1: মোল সংখ্যা বের করো (n = m/M)
পদক্ষেপ 2: লিটারে আয়তন = n × 22.4
```

**উদাহরণ:**
```
2.2g CO₂ এর আয়তন (STP)?
- CO₂ এর আণবিক ভর = 44
- মোল: 2.2 / 44 = 0.05 মোল
- আয়তন: 0.05 × 22.4 = 1.12 লিটার

সরাসরি সূত্র:
V = (m/M) × 22.4 = (2.2/44) × 22.4 = 1.12 L
```

**গুরুত্বপূর্ণ মনে রাখার বিষয়:**
```
✓ শুধুমাত্র গ্যাসের জন্য প্রযোজ্য
✓ শুধুমাত্র STP-তে প্রযোজ্য (0°C, 1 atm)
✗ তরল বা কঠিনের জন্য নয়
✗ অন্য তাপমাত্রা/চাপে নয়
```

## ৪. আণবিক ভর (STP থেকে): M = (m × 22.4) / V

### অজানা পদার্থ চিনে পাওয়া

**সমস্যা:** আমি একটি নতুন গ্যাসের ভর এবং আয়তন জানি, কিন্তু এটি কী?

**সমাধান:** আণবিক ভর খুঁজুন - এটি পদার্থকে পরিচয় দেয়!

**কীভাবে কাজ করে?**
```
আমরা জানি:
- n = V / 22.4 (STP-তে আয়তন থেকে মোল)
- M = m / n (মোল থেকে আণবিক ভর)

একত্রিত করলে:
M = m / (V/22.4)
  = m × 22.4 / V
```

**ব্যবহার:**
```
পদক্ষেপ 1: গ্যাসের ভর জানো
পদক্ষেপ 2: STP-তে এর আয়তন জানো
পদক্ষেপ 3: গুণ করো: ভর × 22.4, তারপর ভাগ করো আয়তন দিয়ে
```

**উদাহরণ:**
```
একটি গ্যাস: 6.4g ভর, 2.24L আয়তন (STP)

পদ্ধতি 1 (দুই ধাপ):
- মোল = 2.24 / 22.4 = 0.1
- আণবিক ভর = 6.4 / 0.1 = 64 g/mol

পদ্ধতি 2 (সরাসরি):
- M = (6.4 × 22.4) / 2.24 = 64 g/mol

পদার্থ চিনে পেতে: M=64 মানে সম্ভবত HNO₃ বা চর্বি
```

## ৫. পরমাণুর সংখ্যা

### অণু থেকে পরমাণু

**পার্থক্য বুঝি:**
```
অণু = ক্ষুদ্রতম ইউনিট যা যৌগের ধর্ম রাখে
পরমাণু = পরমাণু দিয়ে তৈরি

উদাহরণ:
- 1 CO₂ অণু = 3টি পরমাণু (1C + 2O)
- 1 NH₃ অণু = 4টি পরমাণু (1N + 3H)
- 1 H₂SO₄ অণু = 7টি পরমাণু (2H + 1S + 4O)
```

**সূত্র:**
```
মোট পরমাণু = অণুর সংখ্যা × প্রতিটি অণুতে পরমাণুর সংখ্যা

যদি:
- অণুর সংখ্যা = 3.01 × 10²³
- প্রতিটি অণুতে 4টি পরমাণু (NH₃)
- মোট পরমাণু = 3.01 × 10²³ × 4 = 1.20 × 10²⁴
```

**উদাহরণ:**
```
11.2L NH₃ গ্যাস (STP)

ধাপ 1: মোল
- n = 11.2 / 22.4 = 0.5 মোল

ধাপ 2: অণুর সংখ্যা
- N = 0.5 × 6.022×10²³ = 3.011×10²³ অণু

ধাপ 3: পরমাণুর সংখ্যা (NH₃ তে 4 পরমাণু)
- মোট পরমাণু = 3.011×10²³ × 4 = 1.2044×10²⁴ পরমাণু
```

## ৬. ঘনত্ব থেকে ভর: m = ρ × V

### তরল পদার্থ নিয়ে কাজ

**ঘনত্ব কি?**
```
ঘনত্ব = প্রতি একক আয়তনে ভর
ρ (রো) = m / V

তাই:
m = ρ × V
```

**সাধারণ ঘনত্ব:**
```
জল: 1 g/mL (সবচেয়ে সহজ মনে রাখা)
ওয়াইন: ~0.99 g/mL
অ্যালকোহল: ~0.79 g/mL
পেট্রল: ~0.75 g/mL
মার্কারি: ~13.5 g/mL
```

**ব্যবহার:**
```
পদক্ষেপ 1: ঘনত্ব জানো
পদক্ষেপ 2: আয়তন জানো
পদক্ষেপ 3: গুণ করো: ভর = ঘনত্ব × আয়তন
```

**উদাহরণ:**
```
3.6 mL জল এ কত অণু?

ধাপ 1: ভর খুঁজো
- জলের ঘনত্ব = 1 g/mL
- ভর = 1 × 3.6 = 3.6g

ধাপ 2: মোল খুঁজো
- মোল = 3.6 / 18 = 0.2

ধাপ 3: অণু খুঁজো
- অণু = 0.2 × 6.022×10²³ = 1.2044×10²³
```

# 📋 সমস্যা সমাধানের পদক্ষেপ

## ধাপ-দর-ধাপ পদ্ধতি

### প্রথমে বিশ্লেষণ করুন:

```
প্রশ্ন পড়ার সময় চিহ্নিত করুন:
✓ কী দেওয়া আছে? (m, V, n, ρ)
✓ কী খুঁজতে হবে? (n, N, V, M)
✓ গ্যাস বা তরল/কঠিন?
✓ STP শর্ত আছে?
```

### তারপর সূত্র বেছে নিন:

| পরিস্থিতি | ব্যবহার করুন | সূত্র |
|----------|-------------|--------|
| ভর → মোল | n = m/M | ভর দ্বারা আণবিক ভর ভাগ করুন |
| মোল → অণু | N = n × Nₐ | মোল 6.022×10²³ দ্বারা গুণ করুন |
| মোল → আয়তন (গ্যাস, STP) | V = n × 22.4 | মোল 22.4 দ্বারা গুণ করুন |
| ভর → অণু (সরাসরি) | N = (m/M) × Nₐ | একটি সূত্রে ভর থেকে অণু |
| আয়তন + ভর → আণবিক ভর | M = (m × 22.4)/V | STP-তে পদার্থ চিনুন |
| তরল এর ভর | m = ρ × V | ঘনত্ব এবং আয়তন গুণ করুন |

# 🎯 চূড়ান্ত টিপস

1. **Nₐ = 6.022 × 10²³** - এটি সবসময় মনে রাখুন
2. **22.4 L/mol** - শুধুমাত্র গ্যাসের জন্য, শুধুমাত্র STP-তে
3. **ভর দিয়ে শুরু করুন** - এটি সবসময় মোল দিয়ে যায়
4. **অণু এবং পরমাণু বিভ্রান্ত করবেন না** - পার্থক্য মনে রাখুন
5. **একক পরীক্ষা করুন** - গণিত সঠিক কিন্তু ইউনিট ভুল হলে উত্তর ভুল
